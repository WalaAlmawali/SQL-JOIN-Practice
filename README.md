# SQL-JOIN-Practice

 Task Description: SQL JOIN Practice
In this task, you will practice how to combine data from multiple tables using different types of SQL JOIN operations.
You are provided with several related database tables (such as Students, Classes, Employees, Orders, and others). These tables represent real-world relationships between data.
Your job is to write SQL queries that correctly connect these tables and retrieve meaningful information.
You will practice the following JOIN types:
 INNER JOIN (matching records only) 
 LEFT JOIN (all records from the left table) 
 RIGHT JOIN (all records from the right table) 
 FULL OUTER JOIN (all records from both tables) 
 SELF JOIN (table joined with itself) 
 CROSS JOIN (all possible combinations) 
Each question will require you to:
 Identify the correct tables to use 
 Choose the correct JOIN type 
 Match tables using appropriate keys 
 Display the required columns only 
 Handle missing values when necessary 
This exercise will help you understand how relational databases connect data and how different JOIN types affect query results.
